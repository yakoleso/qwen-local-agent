# Qwen Agent на GitPod (бесплатно, без paywall)

✅ Работает на 8 ГБ RAM  
✅ Никаких подписок  
✅ Перезапускается автоматически через GitHub Actions  

## Как запустить:
1. Открой в GitPod: https://gitpod.io/#https://github.com/ваш-ник/qwen-local-agent
2. Дождись, пока соберётся `llama.cpp` и загрузится модель (~3–5 мин)
3. Сервер будет доступен по ссылке вида: `https://8080-<hash>.ws-euXX.gitpod.io`
4. Тестовый запрос уже выполнен в терминале

> ⚠️ GitPod убивает сессию через 50 часов. Чтобы держать агент живым — см. `.github/workflows/restart.yml`
